# Minecraft-Mining-Menu
A Minecraft 1.19 plugin that enables chosen drops from mined stone.
<br /><br />

Command to use:
```java
/drop
```

![UI](./assets/pictures/menu.png)

<br /><br />
Using a pickaxe with Fortune enchantment increase the drop chance.
